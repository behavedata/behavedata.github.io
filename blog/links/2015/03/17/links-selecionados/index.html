Desvendando a culinária Indiana

Um grupo de cientistas identificou o que torna a comida indiana única. A partir de uma análise de milhares de receitas, os pesquisadores descobriram que existe uma tendência de se utilizar ingredientes que não tenham interseção de sabores.

PostgreSQL + JSON

O PostgreSQL está cada vez mais capaz de armazenar e tratar documentos em formato JSON. Este artigo é um estudo sobre a possibilidade de se utilizar o Postgre como um substituto para o MongoDB. A resposta curta é: o Postgre poderia ser utilizado se não houver necessidade de atualizar os documentos frequentemente.

Apache Spark 1.3

Uma nova versão do Spark foi lançada. Uma das novidades desta versão é a API de DataFrame inspirada nos DataFrames de R. Este será um formato comum para comunicação e interoperabilidade.

Guia de Hardware para Deep Learning

Para se utilizar Deep Learning de forma competitiva é necessário um grande poder computacional. Para auxiliar na escolha do hardware, Tim Dettmers preparou um guia que cobre desde a escolha da placa gráfica até os coolers e monitores.
