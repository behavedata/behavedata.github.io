Desvendando a culinária Indiana

PostgreSQL + JSON

O PostgreSQL está cada vez mais capaz de armazenar e tratar documentos em formato JSON. Este artigo é um estudo sobre a possibilidade de se utilizar o Postgre como um substituto para o MongoDB. A resposta curta é: o Postgre poderia ser utilizado se não houver necessidade de atualizar os documentos frequentemente.
